Common Installation Commands

pip install python-frontmatter Markdown beautifulsoup4 moviepy pydub openai
# For markdown-video, ffmpeg is also required
# brew install ffmpeg (macOS)
# sudo apt install ffmpeg (Debian/Ubuntu)

For Claude Code

  1. Download as ZIP: Download the repository as a ZIP file and extract it.
  2. Upload to Claude: In your Claude Code project, upload the entire claude-obsidian-skills folder.
  3. Reference in Prompts: Reference the SKILL.md within each skill folder.
